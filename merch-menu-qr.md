# Merch Menu QR Code

Use this URL to generate the QR code for flyers and printed materials:

https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=https://stumptown-merch.vercel.app/

Points to: https://stumptown-merch.vercel.app/

To download a high-res PNG, open the URL above in a browser and save the image.
Bump the size param higher (e.g. 1000x1000) for large-format print.
